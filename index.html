Arkadiusz Więcław

In all my games, I create most of the elements myself, using a Wacom tablet and Autodesk SketchBook Pro. I use tools such as Canva, Gimp, Blender. Then I create the animations and program them using prefabricated parts, so I understand the whole process of creating a game quite well.I have excellent knowledge of UI systems in Unity and am proficient with Scriptable objects.
